### GitHub Software Tutorial 
The GitHub project that I cloned was a penetration-testing-toolkit that was developed by the user Sharma-IT. It is a comprehensive cybersecurity tool for automated security assessments and vulnerability scanning.  

### Prerequisites 
- Windows, Linux, Mac 
- Install VS Code - https://code.visualstudio.com/
- Install Python 3 - https://www.python.org/downloads/

### Step-by-Step Instructions 
- Step 1: Open VS Code as an administrator 
- Step 2: Cloned the repository https://github.com/ReddyyZ/astsu.git in VS Code 
- Step 3: Open a new terminal window in VS Code  
- Step 4: Run this command - python3 -m pip install -r requirements.txt. (All requirements will be installed) 
- Step 5: Run this command - python3 install.py (The tool will be installed)
